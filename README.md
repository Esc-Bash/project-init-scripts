# project-init-scripts

Initialization scripts for the Esc Bash projects.

## Available scripts

### `init-esc-bash-project.sh`

Creates a minimal Bash project scaffold in the provided directory (or current directory by default):

- `bin/main.sh`
- `lib/`
- `tests/smoke.sh`
- `README.md` (only if one does not already exist)

Usage:

```bash
./init-esc-bash-project.sh /path/to/project
```
